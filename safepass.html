SafePass

A modern web app that evaluates password strength in real-time and provides visual feedback

Tech Stack

React Next.js CSS Node.js

Description

SafePass is a modern web app that helps users create secure passwords effortlessly. It evaluates password strength in real-time and provides a dynamic visual strength bar, giving instant feedback on how strong or weak a password is. With an intuitive UI, dark mode support, and helpful suggestions, SafePass empowers users to protect their digital accounts with confidence.

Features

  • Real-time password strength detection
  • Strength bar with color indicators.
  • Dark mode toggle.
  • Modern,responsive UI design.
